Deployment

This website is deployed using GitHub Pages. After making changes to your site, follow these steps to deploy:

  1. Make changes to the relevant files (HTML, CSS, JS).
  2. Commit the changes:
    3. git add .
    git commit -m "Updated website content"
    git push origin main
  3. Your changes will be deployed automatically by GitHub Pages.
